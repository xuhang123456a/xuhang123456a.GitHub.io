# 向量 矩阵

点乘：结果为数值

a点乘b 等于a的模乘以b的模乘以二者夹角的余弦值

可以判断当前物体是否朝向另一个物体，大于0则面对，否则背对

叉乘：结果为向量

- 叉乘的结果向量垂直于向量a、b所在的平面
- c的模等于a的模乘以b的模乘以二者夹角的正弦值
- 满足右手法则，不满足交换律，可以使用叉乘结果的正负值来判断向量ab的相对位置，向量b是处于向量a的顺时针方向还是逆时针方向

# 欧拉角和四元数

[【Unity编程】欧拉角与万向节死锁（图文版）](https://blog.csdn.net/andrewfan/article/details/60981437)

总结来说，欧拉角的“万向节死锁”问题，是由于欧拉旋转定义本身造成的。这种围绕选旋转前固定轴的先Z、再X、再Y的旋转操作，与其最终所预期的三个轴向可以旋转的结果并非一定是一对一的映射。某些情况下是多对一的映射，造成一些旋转自由度的缺失，也就是“死锁”。
