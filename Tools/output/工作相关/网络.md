# 网络

[C#开源网络通信库PESocket的使用](https://blog.csdn.net/Keicyyy/article/details/121634910)

[Unity实现服务器：1、Socket通信 - 小小小卷毛的文章 - 知乎](https://zhuanlan.zhihu.com/p/376599128)

1. 创建socket对象
    new Socket()
2. 绑定ip和端口

```
IPAddress
EndPoint
_serverSocket.bind()
```

1. 开始监听客户端的连接请求

```
_serverSocket.Listen(_maxConnect)
_clientSocket = _serverSocket.Accept()
```

1. 连接上了，就可以发送、接收消息

```
_clientSocket.Send(data)
_clientSocket.Receive(data)
```

**重构**

将Client封装成一个类，要有初始化，开始接收，接收回调等方法

将Server封装成一个类，要有启动服务器，关闭服务器，接收信息等方法，并且还要有管理Client
