# 战斗系统

战斗状态机：准备状态，攻击状态，回到原位状态

战斗连击实现：攻击完不回原位，播放下一个action

技能特效先播完还是人先回到原位，两个位置需要理清前后顺序关系

攻击时被反击死亡，当场死亡还是回到原位死亡

附带subAction反击没有处理血量同步
